# Academicdata
Java-based academic management system using OOP. Person.java (parent) with Teacher and Student (child) demonstrates inheritance, encapsulation, and polymorphism for managing academic data.
          🚀 Key Features
             -📂 Manages academic data (students & teachers)
             -🧬 Implements Inheritance for code reusability
             -🔒 Uses Encapsulation to protect data via getters/setters
             -🔁 Demonstrates Polymorphism through method overriding
             -🧱 Clean and simple class-based architecture
         🧠 OOP Concepts Used
             -Encapsulation
                 Data members are kept private and accessed through public methods.
             -Polymorphism
                 Methods are overridden in child classes to provide specific behavior.
             -Inheritance
                 Teacher and Student inherit common properties from Person.
                 
📈 Purpose:
This project is designed for learning and demonstrating fundamental Java and OOP concepts in a simple and extensible manner.
